**WorldOfTravel - Travel Booking Website**
WorldOfTravel is a web application built using .NET and Microsoft SQL Server that allows users to explore, book, and manage their travel arrangements conveniently. It provides features such as flight booking, train booking, user authentication, and admin functionality.

**Features**

User Authentication: Users can sign up and log in to the platform securely.
Flight Booking: Users can search for flights based on their preferences and book tickets.
Train Booking: Users can search for trains and book tickets for their journey.
Admin Dashboard: Admins have access to an admin dashboard to manage flights, trains, and user bookings.


**Technologies Used**

.NET: The web application is developed using the .NET framework.
Microsoft SQL Server: Database management is handled using Microsoft SQL Server for storing user data, flight details, train details, and booking information.
